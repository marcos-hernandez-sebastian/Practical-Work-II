# Practical Work II — Personal Website

Web project developed using HTML5 and CSS for Practical Work II — 
Computer Engineering, Universidad Francisco de Vitoria (UFV).

🔗 **Live Website:** 

## Project Description

This project is a fully responsive personal website structured using semantic HTML5 and styled with custom global CSS. It can be seen as a portfolio that showcases personal information, university degree details, classmates network and a dedicated topic section.

The website consists of the following pages:

- `index.html`: The main landing page with a general introduction and an overview of the contents.
- `public/about.html`: A personal presentation featuring hobbies, pictures of myself, social media links, and a brief Curriculum Vitae.
- `public/topic.html`: A dedicated section developing a technological field of interest.
- `public/net.html`: A directory with valid external links to the published websites of several classmates.
- `public/degree.html`: A detailed overview of the Computer Engineering degree, including an academic timeline.
- `public/fce.html`: An in-depth subpage focused on the "Fundamentals of Computer Engineering" course.
- `public/contact.html`: A completely structured and validated contact form layout (pure HTML maquetation).
